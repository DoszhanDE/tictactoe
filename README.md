# Tic tac toe

## Instructions

Start backend app:
```sh
cd backend
./mvnw package && docker-compose up --build
```

Start frontend app:
```sh
yarn
yarn start
```

- Open two browser windows with address "http://localhost:3000" (one of windows in Incognito mode)
- Auth by usernames
- Create challenge using one of the users
- Join the challenge by another user
- Play
